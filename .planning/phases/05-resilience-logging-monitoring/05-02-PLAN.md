    ```python
    logger = logging.getLogger(__name__)

    DEFAULT_MAXSIZE = 200
    ```

    **BoundedQueue** class extending `asyncio.Queue`:
    - `__init__(self, maxsize: int = DEFAULT_MAXSIZE)` — call `super().__init__(maxsize=maxsize)`
    - `put_nowait(self, item)` — if `self.full()`, call `self.get_nowait()` (drop oldest) and log `"Queue at capacity (%d) — dropped oldest item"` with the queue size. Do NOT call `task_done()` — since no code in this project uses `Queue.join()`, calling `task_done()` after eviction creates a counter imbalance risk (per RESEARCH.md Pitfall 1). After eviction, call `super().put_nowait(item)`.
    - `async put(self, item)` — call `self.put_nowait(item)` (non-blocking, drop-oldest behavior).
    - The `get_nowait()` eviction may raise `asyncio.QueueEmpty` if the queue is empty and full simultaneously (race condition on single-item queue). Wrap eviction in try/except QueueEmpty: pass.

    **DeadLetterQueue** class:
    - `__init__(self)` — initialize `self._queue: asyncio.Queue = asyncio.Queue()`, `self._count: int = 0`
    - `async put(self, item)` — await `self._queue.put(item)`, increment `self._count`, log `"[ERR_DLQ] Item added to DLQ — total: %d"` with `self._count` at WARNING level.
    - `async get(self)` — await `self._queue.get()`, decrement `self._count`, return item.
    - `snapshot(self) -> dict` — return `{"depth": self._queue.qsize(), "total_accumulated": self._count}`.
    - `async retry_all(self, target_queue: asyncio.Queue) -> int` — while not `self._queue.empty()`, get_nowait, await target_queue.put, task_done, increment count. Return total re-queued. Reset `self._count = self._queue.qsize()` at end.
    </action>
