    3. Add new function after the existing code (after the try/except that creates the file handler):

    ```python
    LogLevelsConfig = dict[str, str] | None
    
    def configure_module_levels(levels_config: LogLevelsConfig) -> None:
        """Set per-module log levels from config dict.
        
        Args:
            levels_config: Dict mapping logger names to level strings, e.g.
                          {"src.crawler": "DEBUG", "src.ai_handler": "INFO"}
        """
        if not levels_config:
            return
        for logger_name, level_name in levels_config.items():
            level = getattr(logging, level_name.upper(), None)
            if level is None:
                logging.getLogger(__name__).warning(
                    "Invalid log level '%s' for logger '%s'", level_name, logger_name
                )
                continue
            logging.getLogger(logger_name).setLevel(level)
            logging.getLogger(__name__).info(
                "Set %s log level to %s", logger_name, level_name
            )
    ```

    **IMPORTANT**: The logger used for warnings/info inside `configure_module_levels()` must be `logging.getLogger(__name__)` (i.e., `src.logging_setup` logger) — NOT `root_logger` or a variable that creates circular issues. Do NOT call `logging.getLogger()` inside the module at import time in a way that would fail.

    Add `LogLevelsConfig` type alias at module level for reuse by config.py.
  </action>
