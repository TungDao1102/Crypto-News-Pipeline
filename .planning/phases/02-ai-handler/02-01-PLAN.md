## Tasks

### 1. Add httpx to requirements.txt

Read `src/__init__.py`, `src/models.py`, `requirements.txt` before editing.

**Action:** Add `httpx>=0.27,<1` as a new line in `requirements.txt`.

**Acceptance criteria:**
- `requirements.txt` contains `httpx>=0.27,<1`
- `pip install -r requirements.txt` succeeds
- `python -c "import httpx"` succeeds without error

### 2. Add DraftContent model to src/models.py

Read `src/models.py` before editing.

**Action:** Add a new Pydantic `BaseModel` subclass named `DraftContent` after the existing `RawMessage` class:

```python
from typing import Literal


class DraftContent(BaseModel):
    title_vn: str
    telegram_markdown: str
    binance_square_markdown: str
    status: Literal["pending", "approved", "rejected", "published"] = "pending"
    tags: list[str] = []
```

**Acceptance criteria:**
- `python -c "from src.models import DraftContent; d = DraftContent(title_vn='a', telegram_markdown='b', binance_square_markdown='c')"` succeeds
- `d.status` defaults to `"pending"`
- `d.tags` defaults to `[]`
- Field types match spec: title_vn=`str`, telegram_markdown=`str`, binance_square_markdown=`str`, status=`Literal[...]`, tags=`list[str]`
