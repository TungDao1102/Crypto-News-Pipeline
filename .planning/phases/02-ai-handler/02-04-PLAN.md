# Plan 04: Wire AI Handler into main.py

## Objective

Connect the AI handler consumer to the main entry point alongside the existing crawler. The system now runs both crawler (producer) and AI handler (consumer) concurrently.

## Tasks

### 1. Update AIConsumer to accept channel_tags mapping

Read `src/config.py`, `src/models.py` before editing.

**Action:** In `AIConsumer.__init__`, accept and store a `channel_tags: dict[str, list[str]]` parameter. This maps `channel_username -> tags` for prompt selection. Build this dict in `main.py` from `config.sources`.

### 2. Wire AI handler into main.py

Read `src/main.py`, `src/ai_handler.py`, `src/config.py` before editing.

**Action:** Modify `src/main.py`:
1. Create `result_queue: asyncio.Queue[DraftContent]` alongside the existing raw_queue
2. Create `TokenBucket(capacity=10, refill_rate=2.0)` 
3. Create `httpx.AsyncClient(timeout=httpx.Timeout(30.0, connect=10.0, read=30.0), limits=httpx.Limits(max_connections=10))` as context manager
4. Build `channel_tags` dict from `config.sources`: `{s.channel: s.tags for s in config.sources}`
5. Create `AIConsumer(raw_queue, result_queue, channel_tags, token_bucket, http_client)`
6. Run crawler and AI consumer concurrently via `asyncio.gather(crawler.start(), ai_consumer.start())`
7. On shutdown: call both `crawler.shutdown()` and `ai_consumer.shutdown()`
8. Close `http_client` on shutdown

```python
async def main() -> None:
    # ... config, logging unchanged ...

    raw_queue: asyncio.Queue[RawMessage] = asyncio.Queue()
    result_queue: asyncio.DraftContent = asyncio.Queue()
    
    channel_tags = {s.channel: s.tags for s in config.sources}
    token_bucket = TokenBucket(capacity=10, refill_rate=2.0)
    
    crawler = TelegramCrawler(config, raw_queue)
    
    async with httpx.AsyncClient(
        timeout=httpx.Timeout(30.0, connect=10.0, read=30.0),
        limits=httpx.Limits(max_connections=10),
    ) as http_client:
        ai_consumer = AIConsumer(
            raw_queue=raw_queue,
            result_queue=result_queue,
            channel_tags=channel_tags,
            token_bucket=token_bucket,
            http_client=http_client,
        )
        
        # Signal handlers...
        
        try:
            await asyncio.gather(
                crawler.start(),
                ai_consumer.start(),
            )
        except asyncio.CancelledError:
            pass
        finally:
            await crawler.shutdown()
            await ai_consumer.shutdown()
```

**Acceptance criteria:**
- Both `crawler.start()` and `ai_consumer.start()` run concurrently
- `channel_tags` dict contains entries for all enabled sources
- On Ctrl+C, both crawler and AI consumer shut down gracefully
- No unhandled exceptions during normal operation
